00008  /* 
00009   * The purpose of this file is to help look up character set names.  
00010   * 
00011   * Any given encoding may be known by several (usually similar) aliases.  
00012   * For example, a system using ASCII encoding may report the character set as 
00013   * "ASCII", "US-ASCII", or "ISO646-US", among others.  For details on what your system 
00014   * uses, you may wish to consult the nl_langinfo(3) manual page.  
00015   *
00016   * GNU iconv converts a byte sequence from one encoding to another, but before it can do 
00017   * so, it must be told which is which.  In the list below, the preferred GNU iconv(3) name 
00018   * is on the left and an alias is on the right.  It is a simple exercise, left to the reader, 
00019   * to write a function that uses these data to look up the canonical name when provided 
00020   * an alias.  
00021   */
00022
